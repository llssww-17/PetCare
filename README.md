# :dog: PetCare

<p>반려동물을 소유하고 있는 개인과 케어 서비스를 제공할 수 있는 기업 또는 개인 간에 매칭을 시켜주는 서비스입니다.</p></br>

## :date: 프로젝트 기간
<p>2022년 10월 4일 ~ 10월 18일</p></br>

## :link: 실행 방법
- 터미널1
```
$ webpack
```
- 터미널2
```
$ node .\server\module\Server.js
```
